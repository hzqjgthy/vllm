


（1）
创建虚拟环境：python -m venv venv，在当前文件夹下会出现一个文件夹 venv
python -m venv venv ，创建的虚拟环境版本和当前执行的python版本一致，创建的虚拟环境是干净的，不会包含当前执行环境中已安装的第三方依赖包。


（2）
激活环境：.\venv\Scripts\Activate.ps1


===============================


（3）
退出当前虚拟环境：deactivate
删除当前虚拟环境：Remove-Item -Recurse -Force venv

************************可以不使用虚拟环境，虚拟环境占内存太大，使用conda环境：graphrag***********

（4）
安装依赖：
使用poetry进行安装，需要先安装：pip install poetry
安装依赖：poetry install


（5）初始化
poetry run poe init --root ./

（6）手动配置
配置  .env
配置  settings.yaml


（7）创建索引
poetry run poe index --root ./
这步如果运行不成功：
    # 测试虚拟环境中的 Python
    D:\agent\graphrag\graphrag\venv\Scripts\python.exe --version

    # 测试 GraphRAG 是否可用
    D:\agent\graphrag\graphrag\venv\Scripts\python.exe -m graphrag --help

    # 直接运行索引命令
    D:\agent\graphrag\graphrag\venv\Scripts\python.exe -m graphrag index --root ./


（8）local 检索
poetry run poe query --root ./ --method local --query "苹果公司都有哪些产品？"



（9）global 检索
poetry run poe query --method global --query "数据中有哪些主题"
